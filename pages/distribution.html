Instruções para a Equipe

  1. Escaneie o QR Code ou acesse o link fornecido
  2. Aceite os termos de consentimento para coleta de dados
  3. Clique em "Iniciar Leitura" para começar a coleta
  4. Aguarde 2-3 minutos para coleta de dados suficientes
  5. Clique em "Parar Leitura" quando terminar
  6. Clique em "Enviar Dados" para enviar ao sistema
  7. Repita o processo em diferentes horários e locais
